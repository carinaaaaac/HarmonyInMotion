#  Harmony in Motion: A Musical Exploration

A responsive, educational web project developed as part of an **Erasmus+** program. This website serves as an interactive digital guide dedicated to the art of music and the emotional, cultural, and historical impact of various musical instruments.

##  Features

* **Responsive Web Design (RWD):** The website is built with a mobile-first approach. It uses a custom 12-column CSS grid system (`col-`, `col-s-`) and media queries to ensure the layout adapts seamlessly to mobile phones, tablets, and desktop screens.
* **Modular Styling:** Each page utilizes its own external CSS stylesheet for clean code separation and modularity.
* **Engaging UI/UX:** The user interface features custom styling including CSS linear gradients, fixed background images, circular imagery (`border-radius: 50%`), and interactive hover effects on the sidebar navigation menus.
* **Semantic HTML:** Structured using standard HTML5 tags for accessibility and clarity.

##  Website Structure

The application features a cohesive navigation system linking four main sections:

1. **Home (`music.html`):** An introduction titled "Harmony in Motion," exploring music as a universal language that transcends boundaries and creates unity.
2. **The Piano (`piano.html`):** A section titled "Keys of Eternity" that details the instrument's grandeur, versatility, and ability to convey a full spectrum of human sentiment, from classical to jazz.
3. **The Guitar (`guitar.html`):** A page titled "Strings of Emotion" which describes the guitar as a symbol of passion and rebellion, covering its acoustic warmth and electric energy.
4. **The Ukulele (`ukulele.html`):** Titled "Ukulele Serenade," this section highlights the instrument's cheerful disposition, Hawaiian origins, and accessibility for beginners.

##  Technologies Used

* **HTML5:** For semantic page structure and content.
* **CSS3:** For styling, custom grid layouts, responsive media queries, and visual effects (shadows, gradients, hover states).
